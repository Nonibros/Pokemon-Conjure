Contributors to Pokémon Conjure

Latest update: July 2nd 2016

Credits

Pokémon Conjure is currently mostly produced by Nonibros and Hajima. However, we do have other contributors. See below for a total list:

  • Hajima - For making the idea and most maps so far.
  • Nonibros - Has done insane scripting (most everything) and made the Github and this website. He also made the file structure for the project.
  • Deltakirby - Will be doing/has done overworld sprites in the near future.
  • Jobless Tuber Music - Made some Background Music(s).
  • Game Freak - Made both Firered, which is what we are basing our project on, and Pokémon.
  • JPAN - Credit for the save block hack. And general awesomeness. :)
  • Jambo51 - The Flag Extension hack, without which we could not have gone this far this smoothly.
  • Pokefreak890 - Made the 1020 rombase.
  • HackMew - For a lot of random fixes and XSE, without which we couldn't do anything much.
  • Gamer2020 - PGE and the bug fixed XSE.
  • KarateKid552 - Creating G3HS.
  • Lu-Ho - Creating Advance Map. (As terribly buggy as it is.)
