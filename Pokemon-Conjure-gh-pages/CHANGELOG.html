Pokémon Conjure Change Log

Latest update: June 27th 2016

Introduction

This change log is for the Pokémon Firered hack, Pokémon Conjure by Hajima and Nonibros. The thread containing the details on the project can be found here.

See here for the GitHub project.

A few things

Thank you everyone for your interest in this project! We really appreciate the support we have received so far from the community!

If you have any features or suggestions for the project, PM Nonibros or Hajima on the forums.

If you find any bugs, or there are broken parts of the game, PM Nonibros or Hajima on the forums.

Key Notes

Note

  • 0.0.1 is a special patch. This is the one and only time that one needs to patch it to a clean rom. The rest of the patches will be independent, that is you can patch ANY of them to the 0.0.1 patch and play.

Note

  • The only time we change the X in X.Y.Z in our version number is when something REALLY major happens. This can be either a huge overall graphics improvement, or the addition of some major ASM Code that changes everything. We may also use this for REALLY major events in game where we feel a break point is needed.
  • After major in-game events, such as beating a gym, the increment will be the Y in X.Y.Z.
  • After regular changes like fully developing route one, the increment will be the Z in X.Y.Z.

Notes

Some people have found a easter egg and think it's a glitch or broken game. It's not. To those whom don't know, you might find it.

HINT 1: It's only for the bad side.

HINT 2: It's between Route 1 and Kantohara.

Known Issues

These issues are known by the developers and are being worked on.

  • The first house in Thimble Town has a broken warp that breaks the game.
  • There is currently no healing center nor mom, so when the player loses, the healing doesn't work right. Right now you are healed and you fight Blane as punishment for losing :).

These issues are known by the developers but won't be fixed.

  • When walking to the lab, it's possible for Oak to walk through the old lady. This won't be changed because it is funny. :)

[1.0.2] - 2016-06-27

  • Opened Route 2, though it has not been edited yet.

Fixed

  • Advance Map was having some issues. I filled in all random empty space between 0x1500000 - 0x1504A00.
