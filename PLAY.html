A Simple Guide

This is a simple guide on how to play the Pokemon Firered hack, Pokémon Conjure, by Hajima and Nonibros.

Download Version 0.0.1 Patch Download the current Patch

How to play!

Thanks you for your interest in Pokémon Conjure!

Some people have had issues with patching and understanding what in general to do, so this page is simply a page which explains what tools are needed.

This page is also here so as to not anger any mods on the Pokecommunity whom understandably don't want to get in trouble because of suggesting a link to a website which has a ROM, due to legal trouble and what not.

So, this page was made to get past those rules using a friendly little guide, which is NOT (and I read them very well) against the aforemeantioned rules.

Finally, as I am not with the Pokecommunity, it is my right as a user to create any guide I wish, without adding an illegal ROM file here, and yet be able to point to what you need.

With that said, lets begin.

How to Play

First thing first. You need the ROM. Emuparadise is a fantastic site for any of your ROM needs, and it is where we will be obtaining our ROM.

The link to the website (and ROM) is here.

Next you need NUPS, which is here.

You need to download the 0.0.1 patch and then patch it using nups to the Firered ROM first.

After that is done then you patch the current version patch to your 0.0.1 patched Firered. It will be good to keep the 0.0.1.zip, as any new versions you get must be patch to that version.
It is the base, so to speak, upon which all others are built.

Afterwards

You're done! Get to playing!
